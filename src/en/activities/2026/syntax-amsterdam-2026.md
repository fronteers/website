Wes Bos and Scott Tolinski from [Syntax FM](https://syntax.fm/) are in Amsterdam for JS Nation and React Summit, and are hosting a meetup for web developers.

This is part of the JS Nation opening party, but you don't need a conference ticket for Syntax LIVE.

## Schedule

- 19:00 — doors open
- 19:30 — Syntax LIVE show

## Sign up

Free entry. [Register via syntax.fm/meetup](https://syntax.fm/meetup).

Use promo code **SYNTAX** for 15% off combo tickets for [JS Nation](https://jsnation.com/) and [React Summit](https://reactsummit.com/).
